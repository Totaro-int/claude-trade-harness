SETUP

① 증권사 연결

증권사 OpenAPI 문서 URL과 API 키를 입력하면 Claude가 어댑터를 자동 생성합니다. 키는 이 컴퓨터의 .env 파일에만 저장됩니다.

② 어댑터 생성 중

Claude가 API 문서를 읽고 어댑터 코드를 작성합니다. 생성된 코드는 정적 검사·컴파일·실연결 테스트를 통과해야 활성화됩니다.

③ 연결 테스트

④ 온보딩 완료

어댑터 연결이 끝났습니다. 이 도구는 보유종목을 읽어서 분석만 하며 주문을 실행하지 않습니다.

⚠ 산출되는 분석은 투자 자문이 아니며, 모든 투자 판단과 손실은 사용자 본인 책임입니다. 어떠한 수익도 보장하지 않습니다.
